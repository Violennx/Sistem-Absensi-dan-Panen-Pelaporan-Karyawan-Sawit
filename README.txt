CARA MENGGUNAKAN PROGRAM INI
1. Pastikan untuk mengunduh XAMPP terlebih dahulu
2. Save folder database db_sihaksa dan copy ke C:\xampp\mysql\data 
3. Save semua file dan folder dalam repository ini kecuali db_sihaksa dan dokumentasi
4. Simpan semua file dan folder dalam satu folder baru yaitu Laporan. 
5. Simpan di tempat yang diinginkan
6. Open Project di VSCODE atau IDE lainnya
7. Open folder Laporan yang baru dibuat
8. Run Code Java di file LaporanApplication.java
9. Buka XAMPP dan nyalakan Apache dam MySQL
10. Buka Browser dan Salin link berikut http://localhost:8081/login
11. Jika terjadi kesalaha, biasanya hanya ubah port yang berjalan
